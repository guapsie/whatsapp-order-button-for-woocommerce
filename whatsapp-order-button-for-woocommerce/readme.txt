Transforms your WooCommerce store into a WhatsApp sales channel. Add a direct WhatsApp order button to your products.

== Description ==

WhatsApp Order Button for WooCommerce provides a seamless integration between your store and WhatsApp. By allowing your customers to initiate inquiries or orders directly via WhatsApp, you reduce friction and increase your conversion rates.

This plugin is designed for straightforward product inquiries. It automatically appends the product name to your predefined message when a customer clicks the WhatsApp button on a single product page.

= Features =

Product Page Integration: Automatically injects a stylish, full-width "Order via WhatsApp" button beneath the Add to Cart form on single product pages.

Global Floating Widget: Displays a persistent, aesthetically pleasing WhatsApp floating button across your entire website.

Custom Default Message: Define a standard greeting that customers will send to you.

Zero Bloat: Lightweight, featuring inline SVG icons to ensure lightning-fast load times without relying on external font libraries (No FontAwesome needed).

Privacy First: We do not collect, store, or process any personal data.

= Third-Party or External Services =

To provide the chat functionality without heavy local scripts, this plugin relies on the official WhatsApp URL API scheme (wa.me or api.whatsapp.com).

What it is used for: The plugin generates dynamic links that redirect the user's browser or device to the WhatsApp application or WhatsApp Web.

What data is sent and when: When a user clicks the WhatsApp button, their browser opens a URL containing your predefined message and the product name. This data is passed directly from the user's browser to WhatsApp. No personal data or user tracking is sent from your WordPress server to WhatsApp.

Terms & Privacy: By using the generated links, users interact with WhatsApp's services.

== Installation ==

Upload the plugin folder to the /wp-content/plugins/ directory, or install it directly through the WordPress plugins screen.

Activate the plugin through the 'Plugins' screen in WordPress.

Navigate to the 'WhatsApp Order' menu in your WordPress dashboard to configure your telephone number and default message.

== Frequently Asked Questions ==

= Do I need a WhatsApp Business account? =
No, the plugin works perfectly with both standard WhatsApp and WhatsApp Business accounts. However, a Business account is highly recommended for professional storefronts.

= Will this slow down my website? =
Not at all. The plugin is meticulously coded to be exceptionally lightweight. It uses inline SVG icons rather than heavy font libraries, ensuring optimal performance.
